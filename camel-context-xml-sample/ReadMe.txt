Camel Router Spring Project
===========================

Basic camel project create camel spring archetype.
Use /main/resources/META-INF.spring/camel-context.xml to define the camel context.
File component listen /src/data/ directory.
Use ctl+c to shutdown camel context.

To build this project use

    mvn install

To run this project with Maven use

    mvn camel:run

For more help see the Apache Camel documentation

    http://camel.apache.org/
